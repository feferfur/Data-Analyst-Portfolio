# [Выбор локации для скважины](ML_for_business.ipynb)

# Краткое описание

Проект посвящён выбору региона для разработки новых нефтяных скважин. На основе данных геологоразведки для трёх регионов были построены модели машинного обучения, позволяющие прогнозировать объём запасов нефти в скважинах, а затем выполнена оценка потенциальной прибыли и рисков разработки.

В работе использовались данные о характеристиках скважин (`f0`, `f1`, `f2`) и объёме запасов нефти (`product`) для трёх регионов. В ходе проекта были выполнены подготовка данных, обучение моделей линейной регрессии, оценка качества прогнозов, расчёт точки безубыточности, моделирование прибыли методом Bootstrap, построение доверительных интервалов и оценка рисков для каждого региона.

# Цель проекта

Определить наиболее перспективный регион для бурения новых нефтяных скважин на основе прогнозируемой прибыли и уровня риска.

# Задачи проекта

* Загрузить и изучить данные по трём регионам.
* Проверить данные на наличие пропусков и дубликатов.
* Удалить неинформативные признаки.
* Исследовать распределения признаков.
* Подготовить признаки и целевой признак для моделирования.
* Разделить данные на обучающую и валидационную выборки.
* Обучить модель линейной регрессии для каждого региона.
* Получить прогнозы объёма запасов нефти.
* Рассчитать средний прогнозируемый объём запасов.
* Оценить качество моделей с помощью метрики RMSE.
* Рассчитать минимальный объём запасов, необходимый для безубыточной разработки.
* Оценить потенциальную прибыль по регионам.
* Провести Bootstrap-анализ для оценки прибыли.
* Рассчитать 95%-е доверительные интервалы прибыли.
* Оценить вероятность убытков для каждого региона.
* Выбрать регион с наиболее привлекательным сочетанием прибыли и риска.
