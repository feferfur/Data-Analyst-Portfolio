# [Определение возраста покупателей](customer_age_estimation.ipynb)

# Краткое описание

Проект посвящён разработке модели компьютерного зрения для определения возраста покупателей по фотографии. Решение может использоваться в сетевых супермаркетах для анализа возрастных групп клиентов и контроля продажи товаров с возрастными ограничениями.

В работе использовался набор фотографий людей и соответствующие значения возраста. На этапе исследования были изучены структура данных и распределение возрастов, проанализированы примеры изображений, после чего была обучена сверточная нейронная сеть на основе предобученной архитектуры ResNet50. Качество модели оценивалось с помощью метрики MAE.

## Основные результаты

* Датасет содержит **7591 изображение**.
* Пропуски в данных отсутствуют.
* Целевой признак представлен возрастом людей в диапазоне от **1 до 100 лет**.
* Наиболее часто в выборке встречаются люди в возрасте от **20 до 40 лет**.
* На распределении наблюдаются пики на «юбилейных» возрастах: **25, 30, 40, 50 лет и далее**.
* В наборе данных присутствуют как цветные, так и чёрно-белые фотографии.
* Часть изображений имеет различное кадрирование и рамки.

### Параметры модели

Для обучения использовалась архитектура **ResNet50**, предобученная на датасете ImageNet.

Дополнительно использовались:

* слой GlobalAveragePooling2D;
* полносвязный слой Dense с функцией активации ReLU;
* оптимизатор Adam;
* learning rate = **0.0001**;
* горизонтальное отражение изображений;
* количество эпох обучения — **5**.

### Метрики модели

* MAE на обучающей выборке — **3.9173**.
* MAE на тестовой выборке — **6.7133**.

Модель успешно достигла качества определения возраста с ошибкой около **6.7 года** на тестовой выборке.

## Вывод

Разработанная модель позволяет автоматически определять возраст покупателей по фотографии и может использоваться для анализа клиентской аудитории. Полученные прогнозы позволяют оценивать возрастные категории посетителей без необходимости запрашивать дополнительные данные у клиентов.

Также модель может применяться как вспомогательный инструмент для контроля продажи товаров с возрастными ограничениями. Полученное значение MAE 6.71 года свидетельствует о том, что модель способна достаточно точно оценивать возраст человека по изображению и может использоваться для решения прикладных задач компьютерного зрения.
