[**Выбор локации для скважины**]()

**Цель проекта**

Строю модель для предсказания объема запасов в новых скважинах для выбора наиболее успешной локации

**Задачи проекта**

- Строю модель для каждого региона
- Рассчитываю достаточный объём сырья для безубыточной разработки новой скважины и сравниваю со средним запасом в каждом регионе
- С помощью техники Bootstrap посчитаю риски
- Выбираю подходящий регион для разработки

**Навыки и инструменты**

Python, Pandas, NumPy, Scikit-learn, предобработка данных, машинное обучение


**Итоги**

Обучила модели и получила предсказания для трех регионов. Выбрала предпочтительный регион для разработки на основе рассчитанной прибыли и рисков. Также рассчитала доверительный интервал для выбранного месторождения, который показывает, что разработка скважины окупится даже при неудачном выборе точек
