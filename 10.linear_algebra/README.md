[**Защита персональных данных клиентов**](linear_algebra.ipynb)

**Цель проекта**

Обеспечиваю защиту персональных данных клиентов страховой компании без ухудшения качества работы модели машинного обучения

**Задачи проекта**

- Проанализирую и подготовлю исходные данные
- Обосную возможность шифрования признаков без влияния на предсказания модели
- Реализую алгоритм шифрования и декодирования данных
- Проверю корректность работы модели на зашифрованных данных

**Навыки и инструменты**

Python, NumPy, Pandas, Scikit-learn, предобработка данных, линейная алгебра, машинное обучение

**Итоги**

Разработала и проверила алгоритм шифрования данных на основе обратимых матриц. Доказала, что качество модели не меняется при обучении на зашифрованных данных
