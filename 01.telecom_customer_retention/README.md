# [Отток клиентов телеком-компании](final_project.ipynb)

# Краткое описание

Проект посвящён разработке модели машинного обучения для прогнозирования оттока клиентов телеком-компании «ТелеДом». Цель исследования заключалась в выявлении клиентов, которые с высокой вероятностью расторгнут договор, чтобы компания могла заранее предложить им специальные условия и повысить уровень удержания.

В работе использовались данные о договорах клиентов, персональная информация абонентов, сведения об интернет-услугах и телефонии. В ходе проекта были выполнены загрузка и объединение нескольких источников данных, предобработка признаков, генерация новых признаков, исследовательский анализ данных, корреляционный анализ, подготовка данных для обучения моделей, обучение и сравнение нескольких алгоритмов классификации, а также оценка качества лучшей модели на тестовой выборке.

# Цель проекта

Разработать модель машинного обучения для прогнозирования оттока клиентов телеком-компании.

## Основные результаты

### Результаты предобработки и Feature Engineering

* Пропусков и дубликатов в исходных данных не обнаружено.
* Колонка `TotalCharges` была приведена к числовому типу.
* Для активных клиентов значение `No` в признаке окончания договора было заменено на дату выгрузки данных.
* Были созданы новые признаки:

  * факт оттока клиента;
  * длительность обслуживания клиента;
  * изменение ежемесячных расходов;
  * использование сервисов компании;
  * количество подключённых интернет-услуг.

### Результаты анализа данных

* Большинство клиентов предпочитают ежемесячную форму оплаты.
* Более высокий уровень оттока наблюдается среди клиентов с ежемесячными договорами.
* Клиенты с расходами в диапазоне от 2000 до 4000 чаще покидают компанию.
* Более высокий отток характерен для пенсионеров, клиентов с супругами и клиентов с детьми.
* Электронный расчётный лист и автоматические способы оплаты связаны с повышенным оттоком.
* Клиенты с оптоволоконным интернетом и дополнительными интернет-услугами чаще прекращают пользоваться услугами компании.
* Выявлена высокая корреляция между различными интернет-услугами.
* Обнаружена связь между количеством подключённых услуг и ежемесячными расходами клиентов.

### Результаты моделей

| Модель                 | ROC-AUC (кросс-валидация) |
| ---------------------- | ------------------------: |
| LogisticRegression     |                      0.78 |
| RandomForestClassifier |                      0.81 |
| LGBMClassifier         |                      0.89 |

### Итоговая модель

* Лучший результат показала модель **LGBMClassifier**.
* ROC-AUC на кросс-валидации составил **0.89**.
* ROC-AUC на тестовой выборке составил **0.92**.

### Важность признаков

Наибольшее влияние на прогноз оттока оказали:

* длительность обслуживания клиента;
* общие расходы за весь период обслуживания;
* ежемесячные расходы;
* наличие двухлетнего договора.

## Вывод

В результате исследования была разработана модель прогнозирования оттока клиентов, позволяющая заранее выявлять абонентов с высоким риском расторжения договора. Лучшее качество показала модель LightGBM, достигнув значения ROC-AUC 0.92 на тестовой выборке.

Проведённый анализ выявил факторы, связанные с повышенной вероятностью ухода клиентов: тип договора, длительность обслуживания, уровень расходов, способ оплаты и набор подключённых услуг. Полученные результаты могут использоваться для разработки программ удержания клиентов, персонализированных предложений и повышения лояльности наиболее рискованных сегментов абонентов.
