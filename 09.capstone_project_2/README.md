[**Восстановление золота из руды**](capstone_project_2.ipynb)

**Цель проекта**

Для компании, разрабатывающей решения для эффективной работы промышленных предприятий подготавливаю прототип модели машинного обучения, которая должна предсказать коэффициент восстановления золота из золотосодержащей руды, чтобы оптимизировать производство и не запускать предприятие с убыточными характеристиками

**Задачи проекта**

- Подготавливаю данные, проверяю расчеты, анализирую признаки выборок
- Исследую:
    - Изменение концентрации металлов в зависимости от этапа производства
    - Распределение размеров гранул исходного сырья на обучающей и тестовой выборке
    - Суммарную концентрацию металлов на разных стадиях
- Создаю функцию для вычисления итоговой метрики sMAPE
- Обучаю модель и оцениваю ее качество кросс-валидацией


**Навыки и инструменты**

Python, Pandas, NumPy, Matplotlib, Scikit-learn, SciPy, предобработка данных, исследовательский анализ данных, машинное обучение


**Итоги**


Проверила сырые исходные данные на корректность, проанализировала и визуализировала данные, обучила модели, выбрала наиболее точную и проверила на адекватность
