# [Определение стоимости автомобилей](used_car_price_prediction.ipynb)

# Краткое описание

Проект посвящён разработке модели для определения рыночной стоимости автомобилей с пробегом для сервиса «Не бит, не крашен». Задача заключалась в построении модели машинного обучения, способной быстро и точно прогнозировать цену автомобиля на основе его технических характеристик и параметров объявления.

В работе использовались данные о подержанных автомобилях, содержащие информацию о цене, типе кузова, модели, марке, мощности двигателя, пробеге, типе топлива, коробке передач, годе регистрации и других характеристиках. В ходе проекта были выполнены предобработка данных, обработка пропусков и аномалий, кодирование категориальных признаков, обучение нескольких моделей машинного обучения, сравнение качества прогнозов, времени обучения и скорости предсказания, а также выбор наиболее эффективной модели.

# Цель проекта

Построить модель машинного обучения для прогнозирования рыночной стоимости автомобиля с наилучшим сочетанием качества предсказания и скорости работы.

## Основные результаты

### Результаты предобработки

* Удалены неинформативные признаки:

  * DateCrawled;
  * RegistrationMonth;
  * DateCreated;
  * NumberOfPictures;
  * PostalCode;
  * LastSeen.
* Пропуски в категориальных признаках заполнены заглушками.
* Удалены дубликаты.
* Из признака `Price` удалено **8904** записей с нулевой стоимостью.
* Из признака `Power` удалено **93** записей с мощностью более 2500 л.с.
* Для **32972** записей с нулевой мощностью выполнена замена на медианное значение по модели автомобиля.
* Удалено **8** записей, для которых не удалось восстановить мощность.
* Из признака `RegistrationYear` удалено **426** записей с годом регистрации до 1960 года и после 2018 года.

### Итоговый результат

* Наилучшее значение RMSE на обучающей выборке показала модель **LGBMRegressor — 1669.68**.
* Допустимые результаты также показали:

  * CatBoostRegressor — **1683.16**;
  * RandomForestRegressor — **1798.71**.
* Самой быстрой среди лучших моделей по времени обучения стала **LGBMRegressor** (**27.3 с**).
* Самой медленной по времени обучения стала **CatBoostRegressor** (**119.3 с**).
* На тестовой выборке модель **LGBMRegressor** показала RMSE **1668.96**.


## Вывод

В результате исследования была разработана модель прогнозирования стоимости подержанных автомобилей, удовлетворяющая требованиям по качеству и скорости работы. Лучшие результаты показала модель LightGBM, обеспечившая минимальное значение RMSE как на обучающей, так и на тестовой выборке.
