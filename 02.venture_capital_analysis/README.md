# [Знакомство с базой данных](basic_sql.ipynb)

# Краткое описание

Проект посвящен работе с базой данных, которая хранит информацию о венчурных фондах и инвестициях в компании-стартапы. Эта база данных основана на датасете Startup Investments, опубликованном на популярной платформе для соревнований по исследованию данных Kaggle. 

# Цель проекта

Проанализировать данные о фондах и инвестициях и написать запросы к базе.

# Решённые задачи

1. Вывела все закрытые компании из таблицы `company`.

2. Получила объём привлечённых инвестиций для новостных компаний США с сортировкой по размеру финансирования.

3. Рассчитала общую сумму сделок по приобретению компаний за наличные в 2011–2013 годах.

4. Нашла пользователей, чьи аккаунты начинаются с `Silver`.

5. Нашла пользователей, чьи аккаунты содержат подстроку `money`, а фамилия начинается на `K`.

6. Рассчитала суммарный объём инвестиций по странам.

7. Определила минимальные и максимальные суммы инвестиций по датам проведения раундов финансирования.

8. Выполнила классификацию фондов по инвестиционной активности (`low_activity`, `middle_activity`, `high_activity`).

9. Рассчитала среднее количество инвестиционных раундов для каждой категории фондов.

10. Проанализировала инвестиционную активность фондов по странам.

11. Сформировала список сотрудников стартапов с информацией об образовании.

12. Рассчитала количество уникальных учебных заведений сотрудников каждой компании.

13. Нашла закрытые компании, у которых первый раунд финансирования одновременно был последним.

14. Получила список сотрудников таких компаний.

15. Сопоставила сотрудников и учебные заведения, которые они окончили.

16. Рассчитала количество учебных заведений для каждого сотрудника выбранных компаний.

17. Рассчитала среднее количество учебных заведений, оконченных сотрудниками закрытых компаний.

18. Рассчитала среднее количество учебных заведений, оконченных сотрудниками компании Facebook.

19. Сопоставила фонды, компании и суммы инвестиций для компаний с более чем шестью важными этапами развития.

20. Проанализировала сделки по приобретению компаний и соотношение стоимости покупки к объёму инвестиций.

21. Получила список социальных компаний (`social`), привлекавших финансирование в 2010–2013 годах.

22. Выполнила помесячный анализ инвестиционных раундов, фондов и сделок по приобретению компаний.

23. Построила сводную таблицу со средними объёмами инвестиций по странам для компаний, основанных в 2011–2013 годах.
