# [Аналитика поездок GoFast](SDA.ipynb)

# Краткое описание

В проекте выполнен статистический анализ данных сервиса аренды самокатов GoFast. Исследование направлено на изучение поведения пользователей, оценку различий между клиентами с подпиской Ultra и пользователями без подписки, а также проверку гипотез, способных помочь бизнесу увеличить выручку и повысить эффективность маркетинговых решений.

Для анализа использовались данные о пользователях сервиса, поездках и тарифах подписки. В ходе работы были выполнены загрузка и предобработка данных, исследовательский анализ пользовательской базы и поездок, объединение таблиц, расчёт помесячной выручки, проверка статистических гипотез, а также решение прикладных задач с использованием биномиального и нормального распределений.

# Цель проекта

Провести статистический анализ данных сервиса GoFast и проверить гипотезы о поведении пользователей и выручке, которые могут способствовать развитию бизнеса.

# Задачи проекта

* Загрузить и изучить данные о пользователях, поездках и тарифах.
* Проверить качество данных и выполнить предобработку.
* Преобразовать типы данных.
* Исследовать распределение пользователей по городам.
* Проанализировать структуру подписок пользователей.
* Изучить возраст пользователей сервиса.
* Исследовать распределение расстояний поездок.
* Проанализировать длительность поездок.
* Выявить аномалии в данных о поездках.
* Объединить данные из нескольких таблиц.
* Разделить пользователей по типу подписки.
* Сравнить характеристики поездок пользователей с подпиской и без подписки.
* Рассчитать помесячную выручку по каждому пользователю.
* Проверить статистические гипотезы:

  * о различии средней длительности поездок;
  * о соответствии среднего расстояния поездки оптимальному значению;
  * о различии помесячной выручки пользователей разных тарифов.
* Сформулировать метод проверки гипотезы для обращений в техподдержку после обновления серверов.
* Решить задачу расчёта количества промокодов для маркетинговой акции.
* Оценить вероятность открытия push-уведомлений с использованием нормальной аппроксимации.
* Сформулировать итоговые выводы исследования.

## Основные результаты

### Пользовательская база

* Количество пользователей в городах присутствия сервиса оказалось сопоставимым.
* Около половины пользователей оформили подписку Ultra.
* Возраст пользователей находится в диапазоне от 12 до 43 лет.
* Средний возраст пользователей составляет около 25 лет.
* Пик распределения возраста приходится примерно на 26 лет.

### Анализ поездок

* Типичная дальность поездки составляет около 3 км.
* Типичная длительность поездки составляет 15–20 минут.
* Обнаружена небольшая группа очень коротких поездок длиной около 600 метров.
* Выявлены поездки с аномально малой длительностью при существенном расстоянии, что может свидетельствовать об ошибках фиксации времени.

### Сравнение пользователей по подписке

* Пользователи с подпиской Ultra в среднем совершают более длительные поездки.
* Пользователи Ultra демонстрируют более стабильные значения дальности поездок.
* Пользователи без подписки чаще совершают как очень короткие, так и очень длинные поездки.
* Поездки с ошибочной длительностью были выявлены только среди пользователей без подписки.

### Помесячная выручка

* Средняя помесячная выручка на пользователя составила около 342 ₽.
* Медианная помесячная выручка составила 319 ₽.
* Распределение выручки имеет правостороннюю асимметрию.
* Стандартное отклонение составило около 154 ₽, что свидетельствует о высокой вариативности дохода.

### Маркетинговые расчёты

* Для выполнения плана по получению не менее 100 продлений подписки вероятность невыполнения которого не превышает 5%, необходимо разослать не менее 1611 промокодов.
* Вероятность того, что push-уведомление откроют не более 399,5 тыс. пользователей из 1 млн получателей, составила 0,1537.

## Результаты проверки гипотез

* Выявлено статистически значимое различие между пользователями с подпиской и без подписки: пользователи с подпиской в среднем тратят на поездки больше времени.
* Не удалось отвергнуть гипотезу о том, что среднее расстояние одной поездки не превышает оптимального значения с точки зрения износа самокатов.
* Выявлено статистически значимое различие между группами пользователей по выручке: средняя помесячная выручка пользователей с подпиской выше, чем пользователей без подписки.
* Для задачи об обращениях в техподдержку после обновления серверов определён корректный метод проверки — односторонний t-тест для зависимых выборок.

# Вывод

Исследование показало, что пользователи подписки Ultra приносят сервису большую выручку и проводят в поездках больше времени, чем пользователи без подписки. Это подтверждает экономическую целесообразность развития и продвижения подписочной модели.

Полученные результаты могут использоваться для планирования маркетинговых кампаний, расчёта эффективности акций по привлечению пользователей к подписке и прогнозирования выручки. Дополнительно выявленные аномалии в данных о длительности поездок указывают на необходимость проверки механизмов фиксации времени поездок для повышения качества данных и точности аналитики.
