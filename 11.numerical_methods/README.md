[**Определение стоимости автомобилей**](numerical_methods.ipynb)

**Цель проекта**

Построю модель машинного обучения для точного и быстрого определения рыночной стоимости автомобилей с пробегом

**Задачи проекта**

- Проведу предобработку и анализ данных
- Обучу и сравню несколько моделей регрессии
- Подберу оптимальные гиперпараметры
- Оценю качество, скорость обучения и предсказания моделей
- Выберу наилучшее решение по требованиям заказчика

**Навыки и инструменты**

Python, Pandas, NumPy, Scikit-learn, CatBoost, LightGBM, предобработка данных, машинное обучение

**Итоги**

Обучила и сравнила несколько моделей. Лучшей признала модель LightGBM, показавшую оптимальный баланс качества и скорости. На тестовой выборке достигла значения RMSE, соответствующего требованиям заказчика.
