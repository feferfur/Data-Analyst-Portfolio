# [Восстановление золота из руды](gold_recovery_optimization.ipynb)

# Краткое описание

Проект посвящён разработке модели машинного обучения для прогнозирования эффективности процесса восстановления золота из золотосодержащей руды. Для решения задачи использовались данные технологического процесса обогащения, содержащие характеристики сырья, промежуточных продуктов и параметров обработки на различных этапах производства.

В ходе работы были выполнены проверка корректности расчёта показателя восстановления, анализ доступных и недоступных признаков, предобработка данных, исследовательский анализ концентрации металлов и размеров гранул сырья, подготовка данных для моделирования, обучение нескольких моделей машинного обучения и выбор лучшей модели по метрике sMAPE.

# Цель проекта

Построить модель машинного обучения для прогнозирования коэффициента восстановления золота на различных этапах технологического процесса обогащения руды.

# Задачи проекта

* Загрузить и изучить обучающую, тестовую и полную выборки данных.
* Проверить корректность расчёта эффективности обогащения.
* Рассчитать значение MAE между фактическими и вычисленными значениями восстановления.

* Проверить данные на наличие дубликатов.
* Исследовать изменение концентрации металлов на различных этапах обработки:

  * золота (Au);
  * серебра (Ag);
  * свинца (Pb).
* Выявить и удалить аномальные нулевые значения концентраций.
* Сравнить распределения размеров гранул сырья в обучающей и тестовой выборках.
* Проверить равенство средних размеров гранул с помощью t-теста.
* Исследовать изменение суммарной концентрации металлов на различных стадиях обработки.
* Восстановить целевые признаки для тестовой выборки.
* Подготовить признаки и целевые переменные для моделирования.
* Реализовать функцию расчёта итоговой метрики sMAPE.
* Обучить и оценить модели:

  * LinearRegression;
  * DecisionTreeRegressor;
  * RandomForestRegressor.
* Подобрать гиперпараметры для моделей дерева решений и случайного леса.
* Сравнить качество моделей по метрике sMAPE.
* Проверить лучшую модель на тестовой выборке.
* Выполнить проверку модели на адекватность с использованием константной модели.

## Основные результаты

* Значение MAE при проверке формулы восстановления оказалось близким к нулю, что подтвердило корректность расчёта эффективности обогащения.
* Пропуски были заполнены методом `ffill`.
* Явные дубликаты в данных не обнаружены.
* Концентрация золота постепенно увеличивается от стадии сырья до финального концентрата.
* Концентрация серебра увеличивается от стадии сырья до стадии чернового концентрата, затем постепенно снижается до финального концентрата.
* Концентрация свинца резко возрастает на стадии чернового концентрата и затем продолжает плавно увеличиваться.
* Обнаружены многочисленные нулевые значения концентраций на промежуточных стадиях обработки, после чего такие наблюдения были удалены.
* Распределения размеров гранул сырья в обучающей и тестовой выборках оказались схожими.
* Суммарная концентрация металлов постепенно увеличивается от сырья к финальному концентрату.

### Результаты моделей

* Линейная регрессия показала итоговое значение sMAPE **8.866318998588971**.
* Решающее дерево показало итоговое значение sMAPE **9.054008921525714**.
* Случайный лес показал итоговое значение sMAPE **8.020906634039864**.

Лучшей моделью признан случайный лес со следующими параметрами:

* `n_estimators = 150`
* `min_samples_split = 12`
* `min_samples_leaf = 20`
* `max_features = 8`
* `max_depth = 5`

На тестовой выборке модель случайного леса показала итоговое значение sMAPE:

* **6.537594367562538**

Результат константной модели:

* sMAPE = **8.47264825132129**

Полученная модель показала более низкое значение ошибки по сравнению с константной моделью.

## Результаты проверки гипотез

* Не удалось отвергнуть гипотезу о том, что средние размеры гранул сырья в обучающей и тестовой выборках равны.
* Полученное p-value составило **0.9233782195495766**, что свидетельствует об отсутствии статистически значимых различий между выборками.
