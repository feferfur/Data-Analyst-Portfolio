# [Модерация комментариев интернет-магазина Викишоп](ML_for_text.ipynb)

# Краткое описание

Проект посвящён разработке модели машинного обучения для автоматического выявления токсичных комментариев в интернет-магазине «Викишоп». Модель предназначена для классификации пользовательских комментариев на токсичные и нетоксичные с целью последующей модерации контента.

В работе использовался датасет `toxic_comments.csv`, содержащий текст комментариев (`text`) и целевой признак токсичности (`toxic`). В ходе проекта были выполнены анализ данных, очистка и лемматизация текстов, подготовка текстовых признаков с использованием методов «мешок слов» и TF-IDF, обучение нескольких моделей классификации, сравнение их качества по метрике F1 и тестирование лучшей модели.

# Цель проекта

Построить модель классификации комментариев, способную выявлять токсичные комментарии с метрикой F1 не ниже 0.75.

## Основные результаты

* В данных не обнаружено пропусков.
* В данных не обнаружено дубликатов.
* Выявлен дисбаланс классов:

  * соотношение классов составляет примерно 9:1;
  * негативных комментариев примерно в 10 раз меньше, чем позитивных.
* Тексты были очищены от лишних символов и лемматизированы.

### Результаты моделей с использованием метода «мешок слов»

* LogisticRegression — F1 ≈ **0.750**
* RandomForestClassifier — F1 ≈ **0.572**
* LGBMClassifier — F1 ≈ **0.760**

### Результаты моделей с использованием TF-IDF

* LogisticRegression — F1 ≈ **0.745**
* RandomForestClassifier — F1 ≈ **0.565**
* LGBMClassifier — F1 ≈ **0.761**

### Лучшая модель

* Лучший результат на обучающей выборке показала модель **LGBMClassifier**, обученная на признаках TF-IDF.
* Значение F1-меры на обучающей выборке составило примерно **0.761**.
* На тестовой выборке модель показала F1-меру примерно **0.762**.
* Требование проекта (**F1 ≥ 0.75**) выполнено.

## Вывод

В результате проекта была разработана модель автоматического определения токсичных комментариев для системы модерации интернет-магазина. Использование такой модели позволяет автоматически выявлять потенциально нежелательный контент и направлять его на дополнительную проверку модераторами.

Наилучшие результаты показала модель LGBMClassifier, обученная на TF-IDF-признаках. Модель достигла F1-меры около 0.762 на тестовой выборке и превысила установленное требование качества. Полученное решение может использоваться как основа для автоматизации процесса модерации пользовательских комментариев.
