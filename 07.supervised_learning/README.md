[**Обучение с учителем**](supervised_learning.ipynb)

**Цель проекта**

Прогнозирую уход клиентов из банка

**Задачи проекта**

- Построить модель машинного обучения с предельно большим значением f1-меры и не менее 0.59
- Изучить и проработать баланс классов

**Навыки и инструменты**

Python, Pandas, Scikit-learn, предобработка данных

**Итоги**

- Изучила баланс классов, он составляет 80:20
- Сбалансировала классы разнвми методами
- Обучила различные модели машинного обучения и выбрала наиболее точную со значением f1-меры, достигающим целевого
